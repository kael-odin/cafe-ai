# Cafe AI

Cafe is a desktop-first AI workspace that brings together chat, tools, files, browser automation, skills, MCP, and digital-human automation in one place.

## Highlights

- Electron + React + Capacitor workspace
- Spaces, conversations, artifacts, canvas, and browser
- App Store / Skills / MCP / Registry extensibility
- AI sources with API key and OAuth login
- Automation apps, message channels, and remote access

## Development

```bash
npm install
npm run dev
```

Typecheck:

```bash
npm run typecheck
```

Unit tests:

```bash
npm run test:unit
```
