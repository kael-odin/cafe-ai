====================================
    Cafe - Installation Guide
====================================

Thank you for downloading Cafe!

-----------------------------------
Installation
-----------------------------------
1. Drag Cafe.app to the "Applications" folder
2. Double-click Cafe.app to launch

-----------------------------------
"Cannot be opened" warning?
-----------------------------------
This is a normal macOS security prompt. Follow these steps:

Option 1 (Recommended):
1. Right-click on Cafe.app
2. Select "Open"
3. Click "Open" in the dialog

Option 2:
1. Open "System Settings" > "Privacy & Security"
2. Scroll down to find the Cafe message
3. Click "Open Anyway"

You only need to do this once.
